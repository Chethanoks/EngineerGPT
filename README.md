# 🤖 EngineerGPT

EngineerGPT is an offline AI assistant designed for engineering students.

## Features

- 💻 Programming & Debugging
- 🤖 AI & Machine Learning Assistance
- 📚 Explain Complex Topics
- 📝 Writing & Editing
- 💡 Brainstorming
- 📋 Project Planning
- 🎯 Interview Preparation
- 🛣️ Learning Roadmaps
- 📊 Data Analysis
- 🧠 Long-Term Memory
- 📄 PDF Question Answering (RAG)
- 🖥️ Fully Offline using Ollama

## Tech Stack

- Python
- Streamlit
- Ollama
- ChromaDB
- Sentence Transformers
- Git & GitHub
